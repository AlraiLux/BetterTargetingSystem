# Better Targeting System

* Go to Dalamud's settings (`/xlsettings`)
* Click on the [Experimental] tab and scroll down to the "Custom Plugin Repositories" section.
* Copy and Paste the following URL on a new line: `https://raw.githubusercontent.com/Lynesth/DalamudPlugins/main/plugins.json`
* Click on the [+] to the right and make sure the [Enabled] box is checked.
* Click on the [Save] icon at the bottom.

Plugin should now show up in the Available Plugins, where you can install it the normal way.
